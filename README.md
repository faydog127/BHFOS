# The Vent Guys - Horizons Web App & CRM

This repository contains the source code for The Vent Guys' main public-facing website and internal CRM system, built with Vite, React, and Supabase.

## Project Overview

- **Public Website:** A fully responsive, SEO-optimized marketing website showcasing services, company information, and lead capture forms.
- **CRM System:** An internal tool for managing leads, customers, and business operations. Includes a dedicated Call Console for sales and support teams.
