# TIS Field Scouting

Mobile-first field scouting tool for multi-unit partner evaluation.

## Project Tracking

- Current state: [STATUS.md](STATUS.md)
- Change history: [CHANGELOG.md](CHANGELOG.md)
- Pricing contract: [pricing-v1-schema-modifier-matrix.md](pricing-v1-schema-modifier-matrix.md)

Update both files when live behavior, validation state, or project phase changes.

## Run

1. `npm install`
2. `npm run dev`

Data is stored locally in SQLite (via sql.js in IndexedDB) and photo blobs live in IndexedDB.
