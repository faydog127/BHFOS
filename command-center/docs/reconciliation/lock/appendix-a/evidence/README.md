# Evidence

Use `EV-YYYY-MM-DD_*` for Appendix A proof artifacts.

Every evidence artifact should include:
- status
- date
- owner
- raw outputs or links
- result summary

If an evidence file is not linked from `../index.md`, it does not count.

Governance note:
- Evidence docs are human-readable indexes; the **SSOT** is the enforced gate + artifacts (`docs/governance/REVIEW_MAP.md`).
- If an evidence doc is cited in a review, also link it from the change’s `review-input.json.artifacts[]` (so the gate submission and the evidence stay connected).
