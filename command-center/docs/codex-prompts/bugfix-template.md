# Codex bugfix template

Bug:
[Describe the failure clearly.]

Expected behavior:
[What should happen.]

Observed behavior:
[What is happening now.]

Likely scope:
[Paths or files to inspect first.]

Constraints:
- preserve existing behavior outside this bug
- prefer surgical changes
- explain root cause, not just symptom fix

Validation:
- reproduce the bug
- apply the fix
- verify the fix
- state any edge cases not tested

Deliverable:
Return:
1. root cause
2. change made
3. files affected
4. risks
5. validation result
